# Анализ аналогов

| Аналог | Что умеет | Где использует ИИ | Сильные стороны | Слабые стороны | Цена | Чем мы лучше |
|--------|-----------|-------------------|----------------|----------------|---------------|---------------------|
| 1. Google Keep OCR | Распознаёт текст с фото заметок | Компьютерное зрение | Быстро, бесплатно, встроен в Android | Плохо распознаёт русский почерк | Бесплатно | Мы специализируемся на русском почерке + исправляем орфографию |
| 2. Яндекс.Почерк | Распознаёт русский рукописный текст | Нейросети | Хорош для русского языка | Закрытая технология, нет API | Бесплатно | У нас открытая модель, можно дообучить |
| 3. ABBYY FineReader | OCR с поддержкой рукописного текста | Компьютерное зрение | Высокая точность | Платная, тяжёлая | Платная (от 5000 ₽) | Бесплатный, лёгкий, работает в боте |
| 4. ChatGPT Vision | Распознаёт текст, исправляет ошибки | Мультимодальная LLM | Понимает контекст | Дорогой, медленный, данные уходят | Платный API | Бесплатный, офлайн, приватный |
| 5. TrOCR | Распознаёт рукописный текст | Трансформер | Бесплатный, можно дообучить | Требует дообучения для русского | Бесплатно | Мы уже дообучили под русский + добавили орфографию |
| 6. Tesseract OCR | Распознаёт печатный и рукописный текст | Классический OCR | Бесплатный, быстрый | Низкая точность на русском почерке | Бесплатно | Наша модель в 4-5 раз точнее |


# Вывод

Наш проект отличается от аналогов:
1. **Бесплатный** — не требует оплаты за API
2. **Офлайн** — работает без интернета
3. **Приватный** — данные не уходят в облако
4. **Специализация на русском почерке** — дообученная модель
5. **Исправление орфографии** — не только распознаёт, но и исправляет ошибки
